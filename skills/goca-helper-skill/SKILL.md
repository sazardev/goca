# Goca Command Helper Skill

## Description
Assists with Goca CLI command generation using the codebase structure:
- internal/domain/ for pure entities
- internal/usecase/ for business logic  
- internal/repository/ for data persistence
- internal/handler/ for HTTP/gRPC adapters
- internal/di/ for dependency injection

The skill provides guidance for:
- Goca feature command to generate Clean Architecture layers
- Goca entity command for domain models
- Goca usecase command for business logic
- Goca repository command for data persistence
- Goca handler command for interface adapters
- Goca di command for dependency injection setup
